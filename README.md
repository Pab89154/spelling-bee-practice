# Spelling Bee Practice

A free spelling bee practice website for grades Pre-K through 12th. Pick your grade, hear the word, spell it, and earn points to buy hints.

## How to use

1. Open `index.html` in any web browser (Chrome, Safari, or Firefox).
2. Choose your grade level.
3. Click **Start practicing!** and spell 10 words per round.
4. Earn 1 point for each correct word. Spend 5 points to buy a hint.

## Features

- Grade levels from Pre-K to 12th grade
- Words get harder as the grade goes up
- Text-to-speech reads each word aloud
- Points and hint shop (5 points per hint)
- 10 words per round with results at the end

## No install needed

This is a single HTML file — no server, no sign-up, no downloads required.
